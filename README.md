# 가계부 백엔드 서버

가계부 애플리케이션을 위한 백엔드 서버.

# 기술 스택

언어: Java 17

프레임워크: Spring Boot

데이터베이스: MariaDB

ORM: JPA (Hibernate)

보안: Spring Security, JWT (JSON Web Token)

배포: 클라우드타입


# 구현 기능

사용자 로그인 및 회원가입 (JWT 기반)

수입/지출 내역 기록 및 조회

월별 고정 지출 및 저축 내역 관리

카테고리별 지출 내역 저장 (식비, 생활비, 기타)

연간 데이터 요약 제공

API를 통한 프론트엔드와 통신
